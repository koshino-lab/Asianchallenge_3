## flutter_web
ビルドコマンド:
- chromeの場合
```
flutter run -d chrome
```
- ローカルサーバを立ててアクセスする場合
  
以下のコマンドを実行し、[http://0.0.0.0:8000](http://0.0.0.0:8000)　にアクセス（スマホなどで試す場合は0.0.0.0にmacのIPアドレスを入れる）
```
flutter run -d web-server --web-port=
8000 --web-hostname=0.0.0.0
```
